# 🎨 ارسمها! — لعبة رسم وتخمين عربية

نسخة عربية بالكامل من لعبة skribbl.io، بنفس القواعد: لاعب يرسم كلمة والباقون يخمنونها في الشات، والأسرع يحصل على نقاط أكثر.

## المميزات

### اللعب
- واجهة عربية كاملة (RTL) تعمل على الجوال والكمبيوتر
- غرف خاصة برمز من 5 أرقام (حتى 12 لاعب)
- ثلاثة أوضاع لعب:
  - **كلاسيكي**: واحد يرسم والباقي يخمن
  - **فريقين 🔴🔵**: توزيع عشوائي على فريقين، والفريق الأعلى نقاطًا يفوز
  - **الكل يرسم 🗳️**: الجميع يرسم نفس الكلمة، ثم تصويت على أفضل رسمة (+100 نقطة لكل صوت)
- تلميحات: تنكشف حروف من الكلمة مع مرور الوقت
- نقاط أكثر للتخمين الأسرع + نقاط للرسام عن كل من خمّن
- تنبيه "قريبة جدًا!" + مقارنة ذكية للعربي (التشكيل والهمزات والتاء المربوطة لا تفرق)
- من خمّن الكلمة يدردش سرًا مع الرسام ومن خمنوا فقط
- بوت تجريبي 🤖 (حتى 4) للعب لوحدك: يخمن ويرسم في دوره ويصوت

### إعدادات الغرفة (للمالك في اللوبي)
- عدد الجولات (1-10) ووقت الرسم (40-120 ثانية)
- فئة الكلمات: 12 فئة (حيوانات، أكل، مهن...) أو الكل — 354 كلمة
- كلمات مخصصة: اكتب كلماتك الخاصة (5 على الأقل) وتُستخدم بدل القائمة

### أدوات الرسم
- 12 لون، 3 أحجام، قلم ✏️، خط مستقيم 📏، تعبئة 🪣، ممحاة 🧽، تراجع، مسح الكل

### حسابات وإحصائيات
- تسجيل اختياري (اسم + كلمة مرور) — الضيوف يلعبون عادي
- تُحفظ الانتصارات وعدد الألعاب ومجموع النقاط في `db.json`
- لوحة متصدرين 🏆 لأفضل 10 لاعبين

### أصوات
- نغمات للتخمين الصحيح والقريب، تكتكة آخر 10 ثواني، بداية الدور، ونهاية اللعبة — مع زر كتم 🔇

## التشغيل على جهازك

تحتاج [Node.js](https://nodejs.org) (نسخة 18 أو أحدث):

```bash
cd rassm-ar
npm install
npm start
```

ثم افتح المتصفح على `http://localhost:3000`

لاختبارها مع الأصدقاء على نفس الشبكة، يدخلون على `http://عنوان-جهازك:3000`

## النشر على الإنترنت (مجانًا)

### Render (الأسهل)
1. ارفع المجلد إلى مستودع GitHub
2. سجل في [render.com](https://render.com) → New → Web Service → اربط مستودعك
3. Build Command = `npm install` | Start Command = `npm start`
4. Deploy وشارك الرابط

بدائل: [Railway](https://railway.app) أو [Glitch](https://glitch.com)

> ملاحظة: الخطة المجانية قد "تنام" بعد خمول فأول دخول يتأخر ~30 ثانية.

## قاعدة بيانات دائمة للحسابات (Neon PostgreSQL)

بدون هذه الخطوة تُحفظ الحسابات في `db.json` وتُمسح مع كل إعادة نشر. للحفظ الدائم:

1. سجل في [neon.tech](https://neon.tech) (مجاني، بدون بطاقة)
2. أنشئ مشروعًا جديدًا وانسخ **Connection String** (يبدأ بـ `postgresql://...`)
3. في Render: افتح خدمتك → **Environment** → **Add Environment Variable**:
   - Key: `DATABASE_URL`
   - Value: الرابط اللي نسخته
4. احفظ — يعيد النشر تلقائيًا، وبيطبع في اللوق: `💾 قاعدة البيانات: PostgreSQL (دائمة)`

الجدول يُنشأ تلقائيًا أول تشغيل. محليًا على جهازك تستمر اللعبة على `db.json` بدون أي إعداد.

## تعديل الإعدادات البرمجية

في أعلى `server.js`: `PICK_TIME`، `MAX_PLAYERS`، `MAX_BOTS`، `VOTE_TIME`.
ولإضافة كلمات أو فئات جديدة، عدّل `words.js`.

## الملفات

- `server.js` — السيرفر ومنطق اللعبة والحسابات (Node.js + Socket.io)
- `words.js` — الكلمات مصنفة بفئات
- `public/index.html` — واجهة اللعبة كاملة
- `db.json` — قاعدة بيانات الحسابات (تُنشأ تلقائيًا)
